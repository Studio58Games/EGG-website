Simplicity and efficiency

Numerous features for endless possibilities

Made with the Monogame framework, EGG is a powerful and straightforward engine which doesn't overwhelm users with countless options and modules.
Create the 2D game of your dream with an enjoyable and fluid workflow.

Simple and plentiful.
EGG offers a rich set of features, from audio files management, to level building and shaders integration. It can be used to create any type of 2D game, and allows exports to a lot of different media.

Prefabs

Design once, use multiple times

Create efficiently.

Assign a texture, hitbox and behaviour to a GameObject and use it as many times as you want. Modify it and all its instances will be updated accordingly.

Prefab instances can also be modified after having being placed, to create in the end slightly different entities.
